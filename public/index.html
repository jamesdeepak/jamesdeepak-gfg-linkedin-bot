📋 Step-by-Step: Create LinkedIn App

1
Go to LinkedIn Developer Portal

Open linkedin.com/developers → Click Create App

Fill in: App name (e.g. "GFG Bot"), your LinkedIn profile, upload any logo → Click Create

2
Request "Share on LinkedIn" product

Go to Products tab → Find "Share on LinkedIn" → Click Request Access

It approves instantly (or within 1–2 minutes).

3
Set the Redirect URL

Go to Auth tab → Under OAuth 2.0 settings → Add this exact redirect URL:

http://localhost:3000/auth/linkedin/callback

Make sure OAuth 2.0 scopes include: openid profile w_member_social

4
Copy Client ID and Client Secret

Still on the Auth tab → Copy Client ID and Client Secret → Paste them in Settings → Save

5
Click Connect below

A LinkedIn login popup will open. Authorize it. Done! ✅
